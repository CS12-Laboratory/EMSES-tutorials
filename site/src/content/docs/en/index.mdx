## Overview

This site collects tutorials for running MPIEMSES3D / EMSES on Kyoto University's *camphor* supercomputer.

<CardGrid>
  <Card title="Getting started" icon="rocket">
    The [Quick Start](./quick-start/) walks you through running `dshield*` end-to-end.
  </Card>
  <Card title="Parameters and inputs" icon="document">
    `dshield*` uses `plasma.toml` as the primary input. After editing `[meta.physical]`, run `emu apply plasma.toml` before submitting jobs.
  </Card>
  <Card title="Helper tools" icon="setting">
    `pip install -r requirements.txt` installs MPIEMSES3D itself plus helpers such as `emu`, `inp2toml`, and `emses-cp`.
  </Card>
  <Card title="Advanced examples" icon="open-book">
    The former `advance/` examples have moved to MPIEMSES3D's [`cookbook`](https://github.com/CS12-Laboratory/MPIEMSES3D/tree/main/cookbook).
  </Card>
</CardGrid>
